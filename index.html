Sobre el curso

Este curso de desarrollo web está diseñado para aquellos que desean iniciarse en el mundo de la programación o fortalecer sus habilidades en desarrollo web. Bajo la guía de un experto ingeniero de sistemas con amplia experiencia en frontend, explorarás los conceptos fundamentales y las herramientas esenciales para lograr un sólido conocimiento del desarrollo web de calidad.

¿Por qué inscribirte?

Aprenderás todo sobre HTML

Aprenderás, etiquetas, y formularios. Cómo usar formularios para recoger información en un sitio web, y también abordaremos la importancia de la estructuración de contenido.

Serás maquetador con CSS

Exploraremos cómo crear animaciones, posicionamiento de los elementos, agregar transiciones y aplicar media querys para adaptar la apariencia de tu sitio web a diferentes dispositivos y tamaños de pantalla.

Dominarás las bases de JS

Aprenderás su sintaxis básica, arrays y objetos, igualmente el uso de funciones y la manipulación del DOM, a todo esto se le suma alguna librería para darle vida a tus sitios.
